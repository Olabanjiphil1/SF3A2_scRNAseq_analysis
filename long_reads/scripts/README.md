# Long-read scripts

The long-read module contains SQANTI3, DEXSeq, junction-comparison, cis-element, motif, and target-junction summary workflows.

`09_sf3a2_strict_novel_sp_ef_em_summary.R` is the final target-junction summary script. It summarizes SF3A2-specific strict-novel junctions across SP, EF, and EM and writes full/compact tables plus matched reference and SF3A2 junction FASTA files.
